# 10112024A
Simple math console apps utilizing System.CommandLine

## help

`dotnet --help`

## create

`dotnet new console`

## add package

`dotnet add package System.CommandLine --prerelease`

## build

`dotnet build`

## run

`dotnet run`

## publish

`dotnet publish -o out`

## run publish

`out\Math --help`
